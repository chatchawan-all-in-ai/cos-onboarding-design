วาง Channel Access Token และ Channel Secret จาก LINE Developers Console เพื่อให้บอทของคุณรับและส่งข้อความได้

⚠️
ต้องทำใน LINE Developers Console ก่อน
ขั้นตอนนี้ต้องเข้า LINE Developers Console เพื่อคัดลอกข้อมูล และตั้งค่า Webhook URL ด้วยตัวเอง
  1. ไปที่ developers.line.me → เลือก Messaging API channel
  2. คัดลอก Channel Access Token (ใต้ "Messaging API" tab)
  3. คัดลอก Channel Secret (ใต้ "Basic settings")
  4. วาง Webhook URL ด้านล่างในช่อง Webhook settings
